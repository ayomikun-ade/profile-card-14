# Profile Card

## Overview

This project provides a clean, accessible, and responsive profile card component. It's designed to elegantly display user information, social links, and personal interests, ensuring a smooth experience across various devices and for all users. It solves the common need for a well-structured and user-friendly online identity display without any complex setup.

## Features

- **Accessible Design**: Built with semantic HTML and ARIA attributes for screen reader compatibility and keyboard navigation.
- **Responsive Layout**: Adapts beautifully to mobile, tablet, and desktop screens, ensuring a consistent user experience.
- **Dynamic Time Display**: Shows the current epoch time in milliseconds, updating every second.
- **Configurable User Information**: Easily update user name, bio, avatar, and lists of hobbies and dislikes.
- **Social Media Integration**: Includes clearly linked social profiles (Twitter, GitHub, LinkedIn, Instagram).
- **Testable Elements**: All visible elements are equipped with `data-testid` attributes for robust automated testing.

## Getting Started

### Installation

To get this project up and running locally, just follow these simple steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/ayomikun-ade/profile-card-14.git
    ```
2.  **Navigate to the Project Directory**:
    ```bash
    cd profile-card-14
    ```
3.  **Open in Browser**:
    Open the `index.html` file directly in your preferred web browser. Since it's a static site, no server setup is needed!

### Usage

Once you've opened `index.html` in your browser, you'll immediately see the profile card.
The current epoch time in milliseconds will start updating automatically every second, and you can interact with the social links and view the displayed information. You can easily modify the content by editing the `index.html` file.

| Technology | Description                                         |
